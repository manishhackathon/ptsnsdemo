# **App Name**: Rewa Royals University

## Core Features:

- Landing Page Vector Background: Full-width background showcasing light-colored vector designs related to Rewa dynasty and tribal art.
- Header Logos: Circular logos on either side of the header: left logo displaying 'gi tag haldi in shahdol' image and right logo showcasing tribal society of MP, India.
- NAAC Accreditation Navigation: A navigation bar featuring 'NAAC Accreditation' with a transparent sub-menu to make important accreditation information accessible.
- University Image Slider: Image slider showcasing various university images to highlight key aspects and events.
- Common University Sections: Sections include About Us, Academics, Admissions, and Contact.
- Footer Section: Standard website footer including copyright information, contact details, and social media links.

## Style Guidelines:

- Primary color: Burgundy (#800020) to represent the heritage of Rewa dynasty.
- Background color: Beige (#F5F5DC), providing a warm and neutral backdrop for content display.
- Accent color: Antique Gold (#CD9575) for highlights, buttons, and key information.
- Body font: 'Roboto', a modern sans-serif providing readability for main text. Headline font: 'Lora' A well-balanced serif with roots in calligraphy.
- Landing page uses a full-width base background with vector designs. A secondary background of 85% screen width will center content elements.
- Simple, elegant icons relevant to academic sections, admissions, and university life. Use of antique gold to align with the color scheme.
- Subtle animations for image slider transitions and interactive elements for a smooth user experience.
